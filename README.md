## 🛠️ Tech Stack

- **Vanilla HTML/CSS/JS** — No frameworks or build tools required
- **SVG** — Pure vector graphics, responsive and precise
- **html2canvas + jsPDF** — Client-side image capture and PDF generation (optional)
